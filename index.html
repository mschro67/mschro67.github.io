The structure of HTML

HTML works with tags. You need them for everything: text, links, pictures, etc. There are opened and closed tags: < opened, > closed

Styling (without a style sheet)

If you want to style without a CSS stylesheet, you can do it in the opened tag like <p style="font-family:arial;"></p>.
This is an example

Links

You can create links in HTML with the a tag, href="example.com" in the opened tag and and some text. For example: 'click here'

Frames

To add a Frame around a image on your website, you have to add <p style="border:solid;"> to the opened tag or to the CSS sheet.

Rounded frames

For rounded frames you use border-radius, like this: < style="border:solid;border-radius:5px;">

Aligning text in the center

If you want to have text in the center of a line, you need text-align:center;.
This is an example.

Bold and italic text

With the b or the strong tag you can make text bold and if you want italic text, you use the i or the em tag. Bold text is slightly brighter than normal text.

Changing CSS classes using JavaScript

You can change the classes of an HTML object with JavaScript by modifying the classList from an element that you get by using selectors like getElementByClass() with things like add(), remove() or toggle().
