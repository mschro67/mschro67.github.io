The structure of HTML

HTML works with tags. You need them for everything: text, links, pictures, etc. There are opened and closed tags:
< opened
> closed

Styling (without a style sheet)

If you want to style without a CSS stylesheet, you can add commands in the opened tag like <[tag] style="font-family:arial;>".
This is an example

Links

You can create links in HTML with the a tag, href="example.com" in the opened tag and and some text. For example: 'click here' back to the main site

Images

If you want to add a image to your Website by adding the img tag. You can add src="example.com" in the img tag.

Frames

To add a Frame around a image on your website, you have to add <[tag] style="border:solid;>" in the opened tag or in the CSS sheet. example

Rounded frames

For rounded Frames you use border-radius, like this: [tag]{border:solid;border-radius:5px;} in action this looks like this.

aligning text in the center

If you want to have text in the center of a line, you need [tag]{text-align:center;}. This is an example.

Bold and italic text

With the b or the strong tag you can make text bold and if you want italic text, you use the i or the em tag. Bold text is slightly brighter than normal text.

Changing CSS classes using JavaScript

You can change the classes of an HTML object with JavaScript by modifying the classList from an element that you get by using selectors like getElementByClass() with things like add(), remove() or toggle().
This text switches to bold if you press the button.
